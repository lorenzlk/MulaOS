How It Works

// 1. Set the activation flag before Mula loads window.Mula = { activateSalesEnablements: true }; // 2. Load the Mula SDK // The BootLoader will detect the flag and activate the tool // 3. The tool provides: // - Visual element highlighting on hover // - Widget type selection (TopShelf/SmartScroll) // - Click-to-insert functionality // - Automatic product fetching via window.Mula.boot()

Features

  • Visual Placement: Hover over any element to highlight it
  • Widget Selection: Choose between TopShelf or SmartScroll widgets
  • Click to Insert: Click on highlighted elements to insert widgets
  • Product Fetching: Automatically calls window.Mula.boot() to fetch products
  • Clean UI: Professional interface with status messages
  • Memory Management: Proper cleanup when deactivated
