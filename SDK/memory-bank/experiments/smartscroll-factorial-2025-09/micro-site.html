üß™ Experimental Design

Design Type

2x2 Factorial Design - Tests two factors simultaneously with all combinations

Factors

1. Layout Factor

  • Control: Current horizontal card layout (4:3 aspect ratio)
  • Treatment: New vertical card layout (2:3 aspect ratio)

2. Monetization Factor

  • Control: Mula monetization (existing affiliate system)
  • Treatment: RevContent monetization (new partner)

Traffic Allocation

  • Equal Split: 25% per variant (2,500 impressions/day per cell)
  • Total Traffic: ~10,000 viewable impressions/day on on3.com
  • Publisher Scope: on3.com only (www.on3.com primary subdomain)

Statistical Power Analysis

Baseline Assumptions

  • Daily Impressions: ~10,000 viewable impressions
  • Baseline CTR: ~1.0% (100 clicks/day total)
  • Per Cell Traffic: ~2,500 impressions/day

Minimum Detectable Effect (MDE)

  • Target MDE: +20% lift (CTR from 1.0% ‚Üí 1.2%)
  • Statistical Power: 80%
  • Significance Level: Œ± = 0.05

Expected Runtime

  • Main Effects: ~18-23 days (3+ weeks)
  • Interaction Effects: ~20-26 days (3-4 weeks)
  • Conservative Estimate: 4 weeks for full analysis

Experimental Hypotheses

Main Effects

  1. Layout Effect (H1): Vertical layout will improve CTR vs horizontal
  2. Monetization Effect (H2): RevContent will improve RPS vs Mula

Interaction Effects

  1. Layout √ó Monetization (H3): Vertical layout + RevContent will have synergistic effects

üìê MDE Assumptions

Baseline Scale

  • ~10,000 viewable impressions/day
  • ~100 clicks/day total ‚áí ~1.0% CTR baseline
  • Split evenly into 4 cells ‚áí ~2,500 impressions/day per cell

A/B Test (One Variant vs Control)

  • MDE = +20% (CTR from 1.0% ‚Üí 1.2% or RPM equivalent)
  • 80% power, Œ±=0.05
  • Sample needed: ~42.7k impressions per arm
  • Runtime: ~9 days total (with 5k impressions/arm/day)
